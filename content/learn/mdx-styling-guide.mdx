### Large Header (H2)

```markdown
## This creates a cyan header with underline
```

**Result:** Large cyan text (text-cyan-400) with bottom border

### Medium Header (H3)

```markdown
### This creates an emerald green header
```

**Result:** Medium emerald text (text-emerald-400)

</SectionCard>

<SectionCard title="üìö Best Practices">

### Structure Your Content

```markdown
## Main Topic

Brief introduction to the topic.

### Subtopic

More detailed information.

**Key points** in _italics_ with `code examples`.

> Important quotes or highlights
```

### Use Components

```markdown
<Warning>
  Important warnings or notes
</Warning>

<SectionCard title="Organized Section">
  Group related content together
</SectionCard>
```

### Code Examples

Always include language for syntax highlighting:

````markdown
```js
// JavaScript code
```

```ts
// TypeScript code
```

```bash
# Terminal commands
```
````

</SectionCard>

This styling creates a **modern, professional** appearance optimized for dark backgrounds with excellent readability and visual hierarchy!
