<SectionCard title="What is MongoDB?">

- A NoSQL database that stores data in flexible, JSON-like documents
- Developed and maintained by MongoDB, Inc.
- Supports dynamic schemas, making it easy to evolve your data model
- Provides powerful querying and indexing capabilities
- Enhances code readability and maintainability

</SectionCard>

<SectionCard title="Install MongoDB Locally">

- Go to [MongoDB Community Server](https://www.mongodb.com/try/download/community) and download the appropriate version for your operating system.
- Go to Tools and download MongoDB Shell and MongoDB Compass, a GUI for MongoDB.

</SectionCard>
